# Remember Me

一个基于 Qt 框架开发的 Todo List 应用，支持任务的分类、排序以及灵活的筛选功能。

## 主要功能

*   **任务管理**：
    *   **创建任务**：支持设置标题、描述、分类、优先级和截止日期。
    *   **编辑任务**：双击任务即可修改详细信息。
    *   **删除任务**：支持批量删除已选中的任务。
    *   **复制任务**：快速复制现有任务，方便创建重复性工作。
*   **视图与组织**：
    *   **多维度排序**：支持按 ID、标题、优先级、截止日期进行升序或降序排列。
    *   **灵活筛选**：
        *   按分类筛选（侧边栏动态分类列表）。
        *   按状态筛选（全部/未完成/已完成）。
        *   按时间筛选（全部/未过期/今年/本月/本周/今天/已过期）。
    *   **搜索**：支持按标题和描述进行实时模糊搜索。
*   **数据持久化**：
    *   使用 SQLite 数据库本地存储，无需配置，自动保存。
    *   数据文件存储在系统标准应用数据目录下，安全可靠。

##  技术栈

*   **语言**: C++
*   **框架**: Qt
*   **构建系统**: CMake
*   **数据库**: SQLite

## 构建与运行

### 环境要求
*   C++ 编译器
*   Qt 6 SDK
*   CMake

### 构建步骤

1.  克隆仓库
    ```bash
    git clone <repository_url>
    cd RememberMe
    ```

2.  创建构建目录并编译
    ```bash
    mkdir build
    cd build
    cmake ..
    cmake --build .
    ```

3.  运行
    在 `build` 目录下找到可执行文件 `remember-me` (Windows 上为 `remember-me.exe`) 并运行。（确保 Qt DLL 在路径中或使用 `windeployqt` 工具）
